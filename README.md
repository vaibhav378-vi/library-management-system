# 📚 Library Management System

A Django-based web application to manage books, students, and borrowing records.

## 🚀 Features
- Add & manage books
- Student management
- Issue & return books
- Dashboard with stats

## 🛠 Tech Stack
- Python
- Django
- SQLite
- HTML, CSS (Bootstrap)

## ▶️ Run Locally
```bash
git clone https://github.com/your-username/library-management-system.git
cd library-management-system
pip install -r requirements.txt
python manage.py runserver
